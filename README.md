# Xani4kaAddonElytra

Аддон для WorldGuard, который добавляет флаг `elytra-block` для контроля использования элитр в регионах.

## Возможности

- Добавляет новый флаг `elytra-block` в WorldGuard
- Блокирует использование элитр в регионах где флаг установлен в `deny`
- Предотвращает надевание элитр в заблокированных регионах
- Останавливает полет на элитрах при входе в заблокированный регион
- Настраиваемые сообщения в конфигурации

## Требования

- Paper/Spigot 1.16.5
- Java 16
- WorldEdit 7.2.0+
- WorldGuard 7.0.4+

## Установка

1. Скачайте jar файл плагина
2. Поместите его в папку `plugins` вашего сервера
3. Убедитесь что WorldEdit и WorldGuard установлены
4. Перезапустите сервер

## Использование

### Команды WorldGuard для управления флагом:

```
# Заблокировать элитры в регионе
/rg flag <регион> elytra-block deny

# Разрешить элитры в регионе (по умолчанию)
/rg flag <регион> elytra-block allow

# Убрать флаг (вернуть к глобальным настройкам)
/rg flag <регион> elytra-block
```

### Примеры:

```
# Заблокировать элитры в регионе spawn
/rg flag spawn elytra-block deny

# Разрешить элитры в регионе pvp
/rg flag pvp elytra-block allow
```

## Конфигурация

Файл `config.yml` позволяет настроить сообщения:

```yaml
messages:
  elytra-blocked: "&cЭлитры заблокированы в этом регионе!"
  elytra-equip-blocked: "&cВы не можете надеть элитры в этом регионе!"
  elytra-use-blocked: "&cИспользование элитр заблокировано в этом регионе!"

debug: false
```

## Как работает

- **allow** (по умолчанию) - элитры работают нормально
- **deny** - элитры заблокированы:
  - Нельзя начать полет на элитрах
  - Нельзя надеть элитры
  - Полет останавливается при входе в регион
  - Показываются настраиваемые сообщения

## Сборка

```bash
mvn clean package
```

Готовый jar файл будет в папке `target/`.
